## 🎯 À propos du projet

**Fit&Zen** est une application web française qui combine fitness et wellness pour accompagner les utilisateurs dans leur quête d'équilibre corps-esprit. Ce repository contient le prototype HTML/CSS/JS de la landing page, conçu pour démontrer l'identité visuelle, l'expérience utilisateur et les interactions clés du produit.

### ✨ Fonctionnalités du prototype

- 🎨 **Design system complet** - Palette teal/dark, typographies harmonieuses
- 📱 **Interface responsive** - Optimisée pour desktop (1440px) & mobile (390px)
- ✨ **Animations fluides** - Micro-interactions et transitions soignées
- 🧭 **Navigation intuitive** - Tunnel de conversion optimisé
- ♿ **Accessibilité** - Respect des standards WCAG
- 🚀 **Performance** - Code optimisé et léger

## 🎨 Design System

### Palette de couleurs

| Couleur | Hex | Usage |
|---------|-----|-------|
| **Teal primaire** | `#1A7A6E` | CTA, éléments interactifs |
| **Teal light** | `#22C4AE` | Accents, hover states |
| **Dark background** | `#0A0F0E` | Fond principal, contraste |

### Typographies

- **Headings** : [Playfair Display](https://fonts.google.com/specimen/Playfair+Display) - Serif élégant pour les titres
- **Body** : [DM Sans](https://fonts.google.com/specimen/DM+Sans) - Sans-serif moderne et lisible

### Layout & Grille

- Grille 12 colonnes
- Largeur maximale : 1440px
- Espacements basés sur un système harmonique (8px base)

## Personas & Recherche utilisateur

Le projet s'appuie sur une recherche utilisateur approfondie avec **2 personas principales** :

### 👤 Léa Aubry
**Profil** : 28 ans, professionnelle urbaine stressée  
**Objectif** : Retrouver équilibre et bien-être malgré un agenda chargé  
**Pain points** : Manque de temps, difficulté à maintenir une routine

### 👤 Marc Dupont
**Profil** : 35 ans, amateur de fitness cherchant à progresser  
**Objectif** : Suivre ses performances et rester motivé  
**Pain points** : Besoin d'accompagnement personnalisé, plateau de progression
